# TaskNest - A Full-Stack Task Management Web App

TaskNest is a full-stack task management application built for the Web Design and Development final project. It allows authenticated users to create, view, update, search, filter, and delete personal tasks.

## Tech Stack

**Frontend:** React, Tailwind CSS, React Router, Axios
**Backend:** Node.js, Express.js
**Database:** MongoDB (MongoDB Atlas)
**Authentication:** JWT (JSON Web Tokens), bcrypt

## Features

- User registration and login (JWT authentication)
- Create, read, update, and delete tasks
- Search tasks by keyword
- Filter tasks by status (Pending / Completed)
- Dashboard with task statistics and completion progress
- Responsive UI

## Project Structure

tasknest/
├── client/    → React frontend (Vite + Tailwind CSS)
└── server/    → Node.js + Express backend (MongoDB Atlas)

## Getting Started

### Backend
cd server
npm install
npm run dev

### Frontend
cd client
npm install
npm run dev
